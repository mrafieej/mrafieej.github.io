Integration

  • Cloud Solution
  • On-Premise Solution
Integrating our cloud solution is as easy as 1, 2, 3!
  1. Create custom alerts using our REST API.
  2. Upload frames into designated S3 buckets or an FTP server of your choice at a 2 FPS rate.
  3. Receive the alerts via a REST call to your desired endpoint(s).

We do not store any of your users' raw data (frames or videos.)

On-Premise Solution

Host locally on your gateway, Linux box or Jetson modules using our containerized software which includes a server with the same API as in our cloud solution. The software can run entirely offline. We support both ARM and x86 architectures.
